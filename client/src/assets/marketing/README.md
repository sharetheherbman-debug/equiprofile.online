# Marketing Assets

This directory contains marketing images used throughout the EquiProfile application.

## Required Images

### 1. Landing Page Hero (`landing-hero.jpg` or `landing-page2.jpg`)
- **Usage**: Main hero image on the landing page (Home.tsx)
- **Requirements**: 
  - High resolution, responsive
  - Suitable for full-width hero section
  - Works well with dark overlay (bg-black/20)
  - Should look good cropped on mobile devices

### 2. Auth Pages Split Screen (`auth-split.jpg`)
- **Usage**: Shared background for Login and Register pages
- **Requirements**: 
  - High quality, professional appearance
  - Suitable for 50/50 split screen layout
  - Works well on the right side of split screen
  - Should complement form on the left

### 3. Training & Scheduling (`training-scheduling.jpg`)
- **Usage**: Features page section for Training & Scheduling
- **Requirements**: 
  - Modern, equine-themed
  - Represents training/scheduling concepts
  - Professional quality

### 4. Reporting & Analytics (`reporting-analytics.jpg`)
- **Usage**: Features page section for Reporting & Analytics
- **Requirements**: 
  - Professional appearance
  - Data visualization or analytics theme
  - Suitable for feature section display

## Image Specifications

- **Format**: JPEG (.jpg)
- **Quality**: High resolution, web-optimized
- **Color**: Natural tones, suitable for dark overlays
- **Theme**: Professional equine/horse management aesthetic

## Deployment

1. Place images in this directory with the exact filenames listed above
2. Images will be bundled with the application during build
3. Import using: `import imageName from '@/assets/marketing/image-name.jpg'`
